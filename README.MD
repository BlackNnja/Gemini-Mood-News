# Gemini 2.5 // Global Omni-Monitor

A cinematic global monitoring dashboard powered by the Gemini 2.5 model.

Features:
- Live AI-generated global event summaries
- Animated typewriter output
- Boot-screen startup animation
- AI mood meter
- Scrolling market/crypto/air traffic tickers
- Multi-language support (ENG, HEB, ARA, SPA, FRE, RUS, JPN, GER)

Upload these files as-is into a GitHub repo, and open `index.html` in a browser.
